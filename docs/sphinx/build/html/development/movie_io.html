In order to read movie data from a format that is not natively supported, update your fire_config.json file to include the path to a python module containing the functions read_movie_meta and read_movie_data.

~/fire/fire_config.json¶
...
{
    "mast": "{fire_dir}/interfaces/mast.py",
    "mast_u": "{fire_dir}/interfaces/mast_u.py",
    "jet": "{fire_dir}/interfaces/jet.py",
    "<my_machine>": "<path_to_my_script>/<my_machine>.py",
    ...
}
...

This function should return a dictionary with the following elments:

movie_meta = {}
